Create Simple QUEUE from PPPoE Secret - Mikrotik Script RouterOS

Script for to create multiple simple queue from pppoe secret account only max-limit


# Agregar Colas simples de lista de pppoe
# Inicia el script

:do {
:local descarga 10M
:local subida 10M
#Extraemo la cantidad de clientes pppoe registrados
:local total [/ppp secret print count-only]
#Se itera desde 0 hasta el total de clientes pppoe
:for x from=0 to=$total do={
#Almacenamos el nombre de usuario
:local nombre [/ppp secret get value-name=name number="$x"]
#Almacenamos la IP
:local ip [/ppp secret get value-name=remote-address number="$x"]
#Creamos la cola simple
/queue simple add name="$nombre" max-limit="$subida/$descarga" target="$ip"
}
}
