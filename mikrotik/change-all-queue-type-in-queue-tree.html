Change all Queue type in Queue Tree - MikroTik Script RouterOS

Very simple how to Change all Queue type in Queue Tree

Change all Queue type to default-small

foreach i in=([/queue tree find]) do={
local cmd ("queue tree set " . $i . " queue=default-small")
:execute $cmd
}
Change all Queue type to default-small

foreach i in=([/queue tree find]) do={
local cmd ("queue tree set " . $i . " queue=default")
:execute $cmd
}
}
Change all Queue Type with custom Queue Type

foreach i in=([/queue tree find]) do={
local cmd ("queue tree set " . $i . " queue=PCQ-DOWN")
:execute $cmd
}
