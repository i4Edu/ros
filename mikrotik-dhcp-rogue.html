HOW TO GET BOT / API / TOKEN TELEGRAM :
----------------------------------------------------
1. Search in your Telegram @BotFather or https://t.me/BotFather
2. Send /newbot
3. Create name like this -> Example: YourName_bot and input again with same name YourName_bot
4. Save your TOKEN / BOT / API -> Example: 5633162xxx:AAFU1dsXcARJlAH_jJmvF1D4Bp8LLTLJjaX
5. Search in your Telegram @YourName_bot and then click START Button

HOT TO GET CHAT ID TELEGRAM :
----------------------------------------------------
1. Search in your Telegram @get_id_bot or https://t.me/get_id_bot
2. Send /start
3. Save your Chat ID like this -> Example: 5537582xxx

Finaly test from your Browser https://api.telegram.org/bot<TOKEN>/sendMessage?chat_id=<CHAT_ID>&text=Hello%20World
SET YOUR MIKROTIK AS SMS SERVER :
--------------------------------------------
1. Make sure the modem has been detected in the MikroTik System > Resources > USB
2. Next check on MikroTik System > Ports to see the channel on the modem.
3. To test send and receive SMS on MikroTik, you can go to the Tools > SMS
